# Проект: Видеостриминг

## О проекте

- **Суть:** Веб-интерфейс площадки видеостриминга.
- **Формат:** Один или несколько HTML-файлов со встроенными (интегрированными) стилями.

## Технологический стек

- HTML5 (семантическая верстка).
- CSS3 (внутри тега `<style>` в заголовке HTML).
- JavaScript (Vanilla JS, внутри тега `<script>` при необходимости).

## Команды (Build & Test)

- **Запуск:** Открытие HTML-файла напрямую в браузере или через расширение Live Server (VS Code).

## Правила кода (Code Guidelines)

- **Архитектура:** Все стили строго внутри HTML-файла в теге `<style>`. Никаких внешних `.css` файлов.
- **Дизайн:** Темная тема (в стиле Netflix), адаптивная сетка (Flexbox/Grid) для постеров и видео.
- **Верстка:** Использовать семантические теги (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`).
- **Медиа:** Использовать заглушки (Placeholder-изображения) для постеров фильмов.

## Формат ответов (Response Style)

- Отвечай строго на русском языке.
- Пиши код сразу целиком (HTML + CSS в одном блоке), чтобы его можно было легко скопировать.
- Будь лаконичен, избегай длинных теоретических объяснений.
