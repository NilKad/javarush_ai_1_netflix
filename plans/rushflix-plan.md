# RushFlix — План реализации

## Файл
`index.html` — один файл, все стили внутри `<style>`, вся логика внутри `<script>`.

## Структура документа

### 1. HTML-скелет (index.html)
- `<!DOCTYPE html>` с русской локалью
- `<meta charset="UTF-8">`, `<meta name="viewport">`
- Внутри `<body>` 4 секции: header, hero, content-rows, footer

### 2. Header
- `position: fixed` сверху, `z-index: 100`
- Изначально `background: transparent`, при скролле (class `scrolled`) → `background: #141414`
- Слева логотип **RushFlix** (жирный, цвет #E50914)
- Справа кнопка "Войти" (outline-стиль)
- Навигация: список ссылок (Главная, Сериалы, Фильмы, Новинки, Мой список)
- Адаптив: на планшетах навигация уменьшается, на мобильных схлопывается

### 3. Hero Banner
- `<section class="hero">`
- Бэкграунд: изображение-заглушка (серый/тёмный фон + текстовая метка "RushFlix Original") через CSS-градиент
- Градиент снизу: `linear-gradient(to top, #141414 0%, transparent 100%)`
- Заголовок фильма (h1)
- Кнопки: ▶ Воспроизвести (белый фон, чёрный текст) и ℹ Подробнее (полупрозрачный фон)
- Краткое описание (макс 2-3 строки)
- Вертикальный padding: ~400-500px

### 4. Content Rows (4 ряда)
- Каждый ряд: заголовок (h2) + карусель постеров
- Карусель: `<div class="row__posters">` с `overflow-x: scroll` и `overflow-y: hidden`
- Постеры: `<img class="row__poster">` с `object-fit: contain`
- Стрелки навигации (влево/вправо) — появляются при наведении на ряд
- Hover на постере: `transform: scale(1.08)`, тень
- **Ряды:**
  1. RushFlix Новинки
  2. В тренде
  3. Популярное
  4. Продолжить просмотр

### 5. Footer
- `<footer class="footer">`
- Иконки соцсетей (через Unicode/символы или псевдоэлементы)
- Сетка ссылок (колонки: Аудиоописание, Справка, Подарочные карты, Вакансии и т.д.)
- Кнопка "Код услуги" (серый outline)
- Копирайт "© 2026 RushFlix, Inc."

### 6. CSS (внутри `<style>`)
- Reset (box-sizing, margin, padding)
- Переменные: `--bg: #141414; --accent: #E50914; --text: #fff; --transition: 0.3s`
- Flexbox + Grid
- Медиа-запросы: desktop-first, брейкпоинт ~768px для планшетов
- Hover-эффекты на постерах и кнопках

### 7. JavaScript (внутри `<script>`, ~50-70 строк)
1. **Header background**: слушатель `scroll` — если `window.scrollY > 50`, добавить класс `scrolled`
2. **Карусели**: стрелки влево/вправо скроллят `.row__posters` на ширину видимой области (`scrollBy({ left: -width, behavior: 'smooth' })`)
3. (Опционально) hover-эффект с информацией о фильме — пока не делаем, только базовый UI

## Этапы реализации

| Этап | Описание |
|------|----------|
| 1 | Написать HTML-скелет: header, hero, 4 ряда, footer |
| 2 | Добавить CSS: layout, цвета, типографика, градиенты |
| 3 | Добавить hover-эффекты и адаптив |
| 4 | Добавить JS: скролл хедера + стрелки каруселей |
| 5 | Финальная проверка и форматирование |

## Используемые изображения
- Постеры — picsum.photos/300/450 или placeholder-серые блоки с текстом
- Hero — CSS-градиент (без внешнего изображения)
