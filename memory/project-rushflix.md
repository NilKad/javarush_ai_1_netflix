# RushFlix — Лендинг кинотеатра

**Тип:** HTML-файл с интегрированными стилями (без внешних файлов)

## Блоки страницы

1. **Header** — логотип RushFlix, навигация (главная, сериалы, фильмы, новинки, мой список), кнопка входа / аватар. Прозрачный, при скролле — тёмный фон.
2. **Hero Banner** — большой фон с фильмом, градиентное затемнение, название, кнопки "Воспроизвести" / "Подробнее", описание. Адаптив.
3. **Content Rows** — 3-4 ряда постеров (Новинки, В тренде, Популярное, Продолжить просмотр). Карусель с горизонтальным скроллом. Hover: scale-увеличение.
4. **Footer** — соцсети (иконки), меню ссылок, кнопка "Код услуги", копирайт.

## Стили
- Тёмная тема: фон #141414, текст белый, акцент #E50914
- Шрифт: Helvetica / Arial / sans-serif
- Градиенты для hero, hover-анимации, адаптив под десктоп и планшеты

## Интерактив (JS)
- Хедер меняет фон при скролле
- Карусели со стрелками влево/вправо
- Hover-эффекты на постерах

## Отличия от Netflix
- Название — RushFlix (не Netflix)
- Визуальный стиль, шрифты и цветовая схема — клон
